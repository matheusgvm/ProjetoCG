Bibliotecas necessárias para iniciar o jogo:
-GCC(compilador)
-Freeglut
-Glew
-Soil
-SDL

Comandos para instalação das bibliotecas:
sudo apt-get install gcc
sudo apt-get install freeglut3-dev
sudo apt-get install libglew-dev
sudo apt-get install libsoil-dev
sudo apt-get install libsdl2-2.0

Instruções:

1)Use o terminal para chegar à pasta do jogo.
2)digite make file no terminal.

-Navegue pelo menu com o mouse
-Use as teclas <a> e <d> para jogar nas fases 1 e 3
-Use as teclas <w>, <a>, <s> e <d> para jogar na fase 2
-<p>:Pausa o jogo
-<r>:Reinicia o jogo
-<esc>:Encerra o jogo
